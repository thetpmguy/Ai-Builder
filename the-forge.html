Scripted mode

Start a session

A candidate is dropped into a deliberately ambiguous KPMG-flavoured problem. The interviewer probes how they scope it, throws an unexpected governance constraint mid-session, and a second agent scores the reasoning — not the prose — against competencies drawn from the job description.

Leave the key blank to run on a built-in scripted fallback — no install, no network, the session always completes. Paste a key to run the interviewer and scorer live. The key stays in this browser tab for the session only; it is never stored and is sent only to api.anthropic.com.
